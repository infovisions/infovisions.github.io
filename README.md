# Information Visions Lab Website

The website for the [Information Visions Lab](https://infovisions.github.io/) at CU Boulder, led by [Evan Peck](https://evanpeck.github.io/) in the [Information Science Department](https://www.colorado.edu/cmci/infoscience).

## Editing the Website

See **[infovisions.github.io](https://infovisions.github.io)** for step-by-step instructions on adding members, projects, publications, news, and more. No coding experience required.

## Local Development

```bash
git clone https://github.com/infovisions/infovisions.github.io.git
cd infovisions.github.io
bundle install
bundle exec jekyll serve --livereload
```
