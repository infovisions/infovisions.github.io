The website for the [Information Visions Lab](https://infovisions.github.io/) at CU Boulder, led by [Evan Peck](https://evanpeck.github.io/) in the [Information Science Department](https://www.colorado.edu/cmci/infoscience).

## Editing the Website

See **[the editing guide](https://github.com/infovisions/infovisions.github.io/blob/main/GUIDE.md)** for step-by-step instructions on adding members, projects, publications, news, and more.
