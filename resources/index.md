# {% include icon.html icon="fa-solid fa-toolbox" %}Resources

While our group publishes academic papers, we also believe in the public mission of improving data communication, through education, design guidelines, or tools. You'll find those resources here. 

{% include section.html %}

## Tools
A set of tiny tools developed by members of our lab to improve data sharing and understanding.

- [Ziptable](http://ziptbl.com/): a tiny tool for quickly sharing small datasets without uploading any data to a server. See the [announcement](https://bsky.app/profile/peck.phd/post/3mhdwbpmrhk2u). [Featured on FlowingData](https://flowingdata.com/2026/04/02/share-small-datasets-stored-in-a-url/)! 
- [📊 Bar Graphs vs Pie Charts 🥧:](https://barvpie.netlify.app/) A tiny tool to demonstrate quick, direct comparisons between bar graphs and pie charts
- [😀 Emojicon Creator 🔢:](https://emojicon-creator.netlify.app/) A tiny tool to create icon arrays with emojis

## Educational Activities 
Learn, adopt, or remix instructional material that our group uses to help people build intuition about the processes or challenges of data visualization

- [🚦 Speed-Limit Colors](./ed/color-speed): Material + instructions for a 1-Hour Classroom Activity on Data + Color
